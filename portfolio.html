Design Principles

Invariants that guide every decision

LLM Boundary

LLM is ONLY used for parsing (D2/D3). All decisions derive from structured signals. This prevents hallucinated recommendations.

Evidence Guarantee

D6 enforcement: every non-missing skill match must carry resume evidence text. Schema violations are rejected or penalised.

Reproducibility

Same resume + same JD → identical score every time. Weights and thresholds are named constants, not magic numbers.

Incremental Safety

Every new feature (capability layer, audit v1.5) is additive and non-breaking. Optional fields default to safe empty values.
